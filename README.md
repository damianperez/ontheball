# Telegram WebApp Debug Studio
## Versión 1.0

Herramienta completa de diagnóstico y comunicación entre:

- Telegram Bot
- Telegram WebApp
- Backend PHP
- Bot API
- Webhook

---

# 1. Requisitos del servidor

## Servidor

Recomendado:

- Apache o Nginx
- HTTPS obligatorio
- PHP 8.1 o superior

Probado con:

- PHP 8.x
